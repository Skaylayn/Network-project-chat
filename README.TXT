What is it:
Server and client exchanges by message


To run server:
    Server.py

To run client:
    Client.py

To stop client:
when "to server: " is written enter 'exit'
In this way server continue its work (waiting a new client)

To stop server:
When "to client: " is written enter 'exit'
When "Would you like to exit?" is written enter 'y'
